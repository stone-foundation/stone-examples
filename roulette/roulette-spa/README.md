# Stone.js - Roulette SPA

Stone.js's Roulette SPA is a single-page application (SPA) built with Stone.js + React and TypeScript, designed to provide a modern and responsive user interface for the Roulette game. It leverages the Stone.js framework for backend services and integrates with the Roulette API.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Bundle for Production

```sh
npm run build
```

### Run the Production build

```sh
npm run preview
```

### Run tests

```sh
npm run test
```

### Run tests with coverage

```sh
npm run test:cvg
```
